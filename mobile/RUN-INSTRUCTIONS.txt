🚀 MODERNERP ANDROID APP ÇALIŞTIRMA
══════════════════════════════════════════════════

✅ Android platformu hazır!
✅ Capacitor sync tamamlandı!

══════════════════════════════════════════════════
📋 ŞIMDI YAPMANIZ GEREKENLER:
══════════════════════════════════════════════════

1️⃣ ANDROID STUDIO'YU MANUEL AÇIN
══════════════════════════════════════════════════

Yöntem 1: Start Menu
   Windows → Android Studio → Aç

Yöntem 2: Dosya Yöneticisi
   Android Studio kurulum klasörü:
   C:\Program Files\Android\Android Studio\bin\studio64.exe
   veya
   %LOCALAPPDATA%\Android\Android Studio\bin\studio64.exe

══════════════════════════════════════════════════

2️⃣ PROJEYI AÇ
══════════════════════════════════════════════════

Android Studio açıldığında:
   
   → "Open an Existing Project" tıklayın
   
   → Bu klasörü seçin:
     C:\ModernERP\Rapor\android
   
   → "OK" tıklayın
   
   → Gradle sync otomatik başlayacak (1-2 dk)

══════════════════════════════════════════════════

3️⃣ İLK KEZ: EMULATOR OLUŞTUR
══════════════════════════════════════════════════

Üst toolbar'da "No Devices" yazıyorsa:

   → Tools → Device Manager
   → "Create Device" butonuna tıklayın
   → Phone → Pixel 5 seçin → Next
   → API 33 (Android 13) seçin → Next
   → Finish

══════════════════════════════════════════════════

4️⃣ UYGULAMAYI ÇALIŞTIR
══════════════════════════════════════════════════

   → Üst toolbar'da cihaz seçin (Pixel 5)
   → Yeşil ▶️ "Run 'app'" butonuna tıklayın
   → Veya Shift + F10
   
   İlk build 3-5 dk sürebilir...

══════════════════════════════════════════════════

5️⃣ NEXT.JS DEV SERVER BAŞLAT (ÖNEMLİ!)
══════════════════════════════════════════════════

YENİ TERMINAL AÇIN:
   
   cd C:\ModernERP\Rapor
   npm run dev

Bu server çalışmadan Android app veri alamaz!

══════════════════════════════════════════════════
🐛 SORUN ÇÖZME
══════════════════════════════════════════════════

❌ "Unable to connect" hatası:
   → npm run dev çalışıyor mu?
   → http://localhost:3000 tarayıcıda açılıyor mu?

❌ White screen / Boş ekran:
   → Chrome'da: chrome://inspect
   → Console'da hata var mı?

❌ Gradle sync hatası:
   → File → Invalidate Caches → Restart
   → Terminal: cd android && .\gradlew clean

❌ Emulator başlamıyor:
   → Tools → SDK Manager
   → SDK Tools → Android Emulator güncelle

══════════════════════════════════════════════════
📱 FİZİKSEL CİHAZ KULLANIYORSANIZ
══════════════════════════════════════════════════

1. Cihazda: Ayarlar → Geliştirici Seçenekleri
2. USB Debugging açın
3. USB ile bilgisayara bağlayın
4. capacitor.config.ts'yi düzenleyin:
   
   url: 'http://192.168.1.XXX:3000',  // PC'nizin IP
   
5. npm run dev yeniden başlatın
6. npx cap sync android
7. Android Studio'da Run
